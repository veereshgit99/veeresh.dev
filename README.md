# veer.dev

Portfolio / personal website built with Next.js (v15.0.2) & Framer Motion

## Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
